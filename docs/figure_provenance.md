# Figure Provenance

This repository includes one command-reproducible public demo figure:

```bash
python scripts/reproduce_public_demo_figure.py --output /tmp/gbm_public_demo_density.svg
```

The generated SVG overlays the analytic harmonic-oscillator reduced density
with the reconstructed reduced density from the public Fourier/form-factor
path.

## Truth Gate

The current public repository does not claim to reproduce paper figures. A
future `scripts/reproduce_paper_figures.py` should be added only if every
figure it generates has public provenance:

- the figure belongs to public or accepted public-facing material;
- all input data are generated by this public repository or are otherwise
  public and cited;
- the script does not require non-public source history, local paths, raw run
  outputs, handoff packets, or unpublished context;
- the figure labels and docs identify exactly which public figure is being
  reproduced.

Until that gate is satisfied, the correct phrase is "public demo figure," not
"paper figure reproduction."
